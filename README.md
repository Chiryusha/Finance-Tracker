# Finance-Tracker
Приложение для отслеживания транзакций, созданное на Angular. Позволяет добавлять доходы и расходы, просматривать историю операций и контролировать общий баланс.

## Функциональность

- Добавление новых транзакций
- Разделение операций на доходы и расходы
- Просмотр списка всех транзакций
- Отображение общего баланса
- Удаление транзакций
- Удобный интерфейс для управления финансами

## Технологии

- Angular
- TypeScript
- HTML
- CSS
